# 🌾 AgroForward

**B2B-маркетплейс форвардных контрактов на сельхозпродукцию.**

Сервис соединяет фермеров и закупщиков напрямую и даёт малому агробизнесу инструменты, которые раньше были доступны только крупным игрокам на бирже: прогноз цен, оценку рисков, умный подбор контрагентов и автоматическую генерацию договоров.

Проект для хакатона **Kodik Launchpad**.

---

## Проблема

Малые и средние фермерские хозяйства в России не имеют доступа к биржевым инструментам хеджирования цены (Московская биржа обслуживает только крупных игроков). Из-за этого фермеры продают урожай по спотовой цене и несут все ценовые риски сами.

## Решение

Внебиржевые (OTC) форвардные контракты через простой веб-сервис:

- **Маркетплейс** предложений (фермеры) и спроса (закупщики)
- **Прогноз цен** на 60 дней (Meta Prophet) с объяснением на человеческом языке (Claude API)
- **Матчинг** — подбор лучших контрагентов с оценкой совместимости
- **Риск-скоринг** сделки по трём осям: цена, контрагент, логистика
- **Бэктест** — честная walk-forward проверка выгоды фиксации цены на исторических данных
- **Генерация договора** в формате DOCX по условиям сделки

## Структура репозитория
## Быстрый старт

### Вариант 1 — Docker Compose (рекомендуется)

```bash
docker-compose up --build
```

- Фронтенд: http://localhost:3000
- API: http://localhost:8000
- Документация API: http://localhost:8000/docs

Для AI-объяснений задайте `ANTHROPIC_API_KEY`. Без ключа прогноз работает в fallback-режиме.

Перед сдачей раскомментируйте блок `frontend` в `docker-compose.yml`.

### Вариант 2 — ручной запуск

Бэкенд:
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python seed_db.py
uvicorn main:app --reload --port 8000
```

Фронтенд (отдельный терминал):
```bash
cd frontend
npm install
npm run dev
```

Откройте http://localhost:3000

## Основные эндпоинты API

| Метод | Путь | Назначение |
|-------|------|-----------|
| GET | /offers | Предложения (продажа) |
| GET | /demands | Спрос (закупка) |
| GET | /forecast?crop=wheat&days=60 | Прогноз + AI-объяснение |
| GET | /match/find?offer_id=1 | Подбор покупателей |
| GET | /risk?offer_id=1&demand_id=1 | Риск-оценка сделки |
| GET | /backtest?crop=wheat | Бэктест фиксации цены |
| GET | /contract/{deal_id}?offer_id=1&demand_id=1 | Договор (DOCX) |
